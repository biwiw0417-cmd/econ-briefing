기준금리 (Base Interest Rate) 2026-07-17

정의 한 나라의 중앙은행(한국의 경우 한국은행)이 경제 상황을 고려하여 결정하는 지표 금리로, 시중 모든 금리의 기준점이 됩니다.

비유 실험 장비로 유입되는 메인 유량 제어 밸브와 같습니다. 이 밸브를 조여서 압력(금리)을 높이면 전체 시스템에 흐르는 유체(돈)의 양이 줄어들고, 밸브를 열면 흐름이 빨라집니다.

기회비용 (Opportunity Cost) 2026-07-18

정의 어떤 선택을 함으로써 포기하게 되는 여러 대안 중 가장 가치가 높은 것의 가치.

비유 한정된 연구실 예산으로 고성능 연산용 컴퓨터를 사기로 결정했다면, 그 돈으로 갈 수 있었던 '해외 학회 참석 기회'나 '최신 분석 장비 구매' 중 가장 가치 있었던 대안 하나가 바로 기회비용이 됩니다.
